# Titan Network

## 概览
自行前往官方文档阅读：[点击这里](https://titannet.gitbook.io/titan-network-cn)

如果还有小朋友不懂的话：

**Titan Network** 是一个数字资源网络，旨在促进社区闲置资源与全球需求相匹配。让每个人都能轻松利用 DePIN* 服务的力量，参与价值互联网的建设。

*DePIN：去中心化物理基础设施网络
